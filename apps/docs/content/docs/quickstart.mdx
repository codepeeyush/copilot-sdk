# Quick Start

Get your AI copilot running in 5 minutes.

<Callout type="info">
**Prerequisites:** Node 18+, React 18+, an LLM API key (OpenAI, Anthropic, etc.)
</Callout>

## Setup

<Steps>
  <Step>
    ### Create API Route (Backend)

    ```ts title="app/api/chat/route.ts"
    import { createRuntime, createOpenAI } from '@yourgpt/llm-sdk';

    const openai = createOpenAI({
      apiKey: process.env.OPENAI_API_KEY!,
    });

    const runtime = createRuntime({
      provider: openai,
      model: 'gpt-4o-mini',
      systemPrompt: 'You are a helpful assistant.',
    });

    export async function POST(request: Request) {
      return runtime.handleRequest(request);
    }
    ```
  </Step>

  <Step>
    ### Add Provider (Frontend)

    ```tsx title="app/providers.tsx"
    'use client';

    import { CopilotProvider } from '@yourgpt/copilot-sdk/react';

    export function Providers({ children }: { children: React.ReactNode }) {
      return (
        <CopilotProvider runtimeUrl="/api/chat">
          {children}
        </CopilotProvider>
      );
    }
    ```

    ```tsx title="app/layout.tsx"
    import { Providers } from './providers';

    export default function RootLayout({ children }) {
      return (
        <html>
          <body>
            <Providers>{children}</Providers>
          </body>
        </html>
      );
    }
    ```
  </Step>

  <Step>
    ### Add Chat Component

    ```tsx title="app/page.tsx"
    import { CopilotChat } from '@yourgpt/copilot-sdk/ui';

    export default function Home() {
      return (
        <div className="h-screen p-4">
          <CopilotChat className="h-full rounded-xl border" />
        </div>
      );
    }
    ```
  </Step>
</Steps>

## Styling (Tailwind CSS v4)

The UI components use Tailwind CSS with [shadcn/ui](https://ui.shadcn.com) theming.

```css title="app/globals.css"
@import "tailwindcss";

/* Include SDK package for Tailwind class detection */
@source "node_modules/@yourgpt/copilot-sdk/src/**/*.{ts,tsx}";

@custom-variant dark (&:is(.dark *));
```

<Callout type="info">
For theming and CSS variables, follow the [shadcn/ui theming guide](https://ui.shadcn.com/docs/theming).
</Callout>
