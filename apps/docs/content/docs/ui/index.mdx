## CopilotChat

The all-in-one chat component. Just drop it in.

```tsx
import { CopilotChat } from '@yourgpt/copilot-sdk-ui';

<CopilotChat
  title="AI Assistant"
  placeholder="Ask me anything..."
  welcomeMessage="Hey! How can I help you today?"
  suggestions={[
    "What can you do?",
    "Help me with something",
  ]}
  className="h-[600px] rounded-xl border"
/>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `"Chat"` | Header title |
| `placeholder` | `string` | `"Type a message..."` | Input placeholder |
| `welcomeMessage` | `string` | - | Initial greeting |
| `suggestions` | `string[]` | `[]` | Quick reply chips |
| `showHeader` | `boolean` | `true` | Show/hide header |
| `showFollowUps` | `boolean` | `true` | Show AI-generated follow-ups |
| `className` | `string` | - | Container classes |

### Aliases

`ConnectedChat` is an alias for `CopilotChat` for backwards compatibility:

```tsx
import { ConnectedChat } from '@yourgpt/copilot-sdk-ui';

// Same as CopilotChat
<ConnectedChat title="Support" />
```

## Chat

Lightweight stateless chat component. You provide the messages.

```tsx
import { Chat } from '@yourgpt/copilot-sdk-ui';

function MyChat() {
  const [messages, setMessages] = useState([]);
  const [input, setInput] = useState('');

  return (
    <Chat
      messages={messages}
      input={input}
      onInputChange={setInput}
      onSubmit={handleSubmit}
      isLoading={isLoading}
    />
  );
}
```

Use this when you need full control over message state.

## DevLogger

Debug panel for development. See the AI's thought process.

```tsx
import { DevLogger } from '@yourgpt/copilot-sdk-ui';
import { useDevLogger } from '@yourgpt/copilot-sdk-react';

function DebugMode() {
  const state = useDevLogger();

  if (process.env.NODE_ENV !== 'development') return null;

  return <DevLogger state={state} position="bottom-left" />;
}
```

Shows:
- Message flow
- Tool calls with inputs/outputs
- Context being sent
- Timing info

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `state` | `DevLoggerState` | required | State from `useDevLogger()` |
| `position` | `string` | `"bottom-right"` | `"bottom-left"`, `"bottom-right"`, `"top-left"`, `"top-right"` |

## Primitives

Build your own UI with these building blocks.

### Markdown

Renders AI responses with syntax highlighting.

```tsx
import { Markdown } from '@yourgpt/copilot-sdk-ui';

<Markdown content={message.content} />
```

### CodeBlock

Syntax-highlighted code with copy button.

```tsx
import { CodeBlock } from '@yourgpt/copilot-sdk-ui';

<CodeBlock
  language="typescript"
  code={`function hello() { return "world"; }`}
/>
```

### MessagePrimitive

Styled message container.

```tsx
import { MessagePrimitive, MessageAvatar, MessageContent } from '@yourgpt/copilot-sdk-ui';

<MessagePrimitive variant={msg.role}>
  <MessageAvatar role={msg.role} />
  <MessageContent>
    <Markdown content={msg.content} />
  </MessageContent>
</MessagePrimitive>
```

### PromptInput

The input area with actions.

```tsx
import {
  PromptInput,
  PromptInputTextarea,
  PromptInputActions,
  PromptInputAction,
  SendIcon,
  StopIcon,
} from '@yourgpt/copilot-sdk-ui';

<PromptInput onSubmit={sendMessage}>
  <PromptInputTextarea placeholder="Type here..." />
  <PromptInputActions>
    {isLoading ? (
      <PromptInputAction onClick={stopGeneration}>
        <StopIcon />
      </PromptInputAction>
    ) : (
      <PromptInputAction type="submit">
        <SendIcon />
      </PromptInputAction>
    )}
  </PromptInputActions>
</PromptInput>
```

### ChatContainer

Scrollable chat container with auto-scroll.

```tsx
import {
  ChatContainerRoot,
  ChatContainerContent,
  ChatContainerScrollAnchor,
  useChatContainer,
} from '@yourgpt/copilot-sdk-ui';

function ChatArea({ messages }) {
  const { containerRef, scrollToBottom } = useChatContainer();

  return (
    <ChatContainerRoot ref={containerRef}>
      <ChatContainerContent>
        {messages.map(msg => (
          <Message key={msg.id} message={msg} />
        ))}
        <ChatContainerScrollAnchor />
      </ChatContainerContent>
    </ChatContainerRoot>
  );
}
```

### ToolSteps

Show tool execution progress.

```tsx
import { ToolSteps, ToolStep, InlineToolSteps } from '@yourgpt/copilot-sdk-ui';

// Detailed view
<ToolSteps>
  <ToolStep
    name="search_products"
    status="completed"
    result={{ found: 5, query: "headphones" }}
  />
  <ToolStep
    name="get_pricing"
    status="running"
  />
</ToolSteps>

// Compact inline view
<InlineToolSteps steps={[
  { name: 'search', status: 'completed' },
  { name: 'process', status: 'running' },
]} />
```

### ToolExecutionMessage

Display tool execution in chat messages.

```tsx
import { ToolExecutionMessage } from '@yourgpt/copilot-sdk-ui';

<ToolExecutionMessage
  execution={{
    id: 'exec-1',
    toolName: 'search_products',
    status: 'completed',
    args: { query: 'headphones' },
    result: { success: true, products: [...] },
  }}
/>
```

### Source


### FollowUpQuestions

AI-generated follow-up suggestions.

```tsx
import { FollowUpQuestions, parseFollowUps } from '@yourgpt/copilot-sdk-ui';

const { cleanContent, followUps } = parseFollowUps(message.content);

{followUps.length > 0 && (
  <FollowUpQuestions
    questions={followUps}
    onSelect={(q) => sendMessage(q)}
  />
)}
```

```tsx
  state="pending"
```

## Styling

Components use CSS variables. Override them in your global CSS:

```css
:root {
  --yourgpt-primary: #6366f1;
  --yourgpt-background: #ffffff;
  --yourgpt-foreground: #171717;
  --yourgpt-muted: #f5f5f5;
  --yourgpt-border: #e5e5e5;
  --yourgpt-radius: 0.5rem;
}

/* Dark mode */
.dark {
  --yourgpt-background: #0a0a0a;
  --yourgpt-foreground: #fafafa;
  --yourgpt-muted: #262626;
  --yourgpt-border: #262626;
}
```

Or use Tailwind classes directly:

```tsx
<CopilotChat
  className="bg-slate-50 dark:bg-slate-900 rounded-2xl shadow-xl"
/>
```

## Custom Message Renderer

Build fully custom message UI:

```tsx
import { useAIChat } from '@yourgpt/copilot-sdk-react';
import {
  Markdown,
  ToolSteps,
  ToolStep,
  FollowUpQuestions,
  parseFollowUps,
  MessagePrimitive,
  MessageAvatar,
  MessageContent,
} from '@yourgpt/copilot-sdk-ui';

function CustomChat() {
  const { messages, sendMessage } = useAIChat();

  return (
    <div className="flex flex-col gap-4">
      {messages.map(msg => {
        const { cleanContent, followUps } = parseFollowUps(msg.content);

        return (
          <MessagePrimitive key={msg.id} variant={msg.role}>
            <MessageAvatar role={msg.role} />
            <MessageContent>
              {/* Tool executions */}
              {msg.toolCalls && msg.toolCalls.length > 0 && (
                <ToolSteps>
                  {msg.toolCalls.map(tc => (
                    <ToolStep key={tc.id} name={tc.name} status={tc.status} />
                  ))}
                </ToolSteps>
              )}

              {/* Message content */}
              <Markdown content={cleanContent} />

              {/* Follow-ups */}
              {followUps.length > 0 && (
                <FollowUpQuestions
                  questions={followUps}
                  onSelect={sendMessage}
                />
              )}
            </MessageContent>
          </MessagePrimitive>
        );
      })}
    </div>
  );
}
```

## Complete Custom Chat Example

```tsx
import { useAIChat } from '@yourgpt/copilot-sdk-react';
import {
  ChatContainerRoot,
  ChatContainerContent,
  ChatContainerScrollAnchor,
  useChatContainer,
  MessagePrimitive,
  MessageAvatar,
  MessageContent,
  Markdown,
  PromptInput,
  PromptInputTextarea,
  PromptInputActions,
  PromptInputAction,
  SendIcon,
  StopIcon,
  Loader,
} from '@yourgpt/copilot-sdk-ui';

function CustomCopilot() {
  const { messages, isLoading, sendMessage, stopGeneration } = useAIChat();
  const { containerRef } = useChatContainer();
  const [input, setInput] = useState('');

  const handleSubmit = () => {
    if (input.trim()) {
      sendMessage(input);
      setInput('');
    }
  };

  return (
    <div className="flex flex-col h-full">
      {/* Messages */}
      <ChatContainerRoot ref={containerRef} className="flex-1">
        <ChatContainerContent>
          {messages.map(msg => (
            <MessagePrimitive key={msg.id} variant={msg.role}>
              <MessageAvatar role={msg.role} />
              <MessageContent>
                <Markdown content={msg.content} />
              </MessageContent>
            </MessagePrimitive>
          ))}
          {isLoading && <Loader />}
          <ChatContainerScrollAnchor />
        </ChatContainerContent>
      </ChatContainerRoot>

      {/* Input */}
      <PromptInput onSubmit={handleSubmit}>
        <PromptInputTextarea
          value={input}
          onChange={(e) => setInput(e.target.value)}
          placeholder="Type a message..."
        />
        <PromptInputActions>
          {isLoading ? (
            <PromptInputAction onClick={stopGeneration}>
              <StopIcon />
            </PromptInputAction>
          ) : (
            <PromptInputAction type="submit">
              <SendIcon />
            </PromptInputAction>
          )}
        </PromptInputActions>
      </PromptInput>
    </div>
  );
}
```
