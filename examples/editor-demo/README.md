# Copilot SDK Showcase

Real-world demo applications showcasing the Copilot SDK capabilities.

## Demos

### Shopping Assistant

E-commerce copilot with product catalog, cart management, and checkout flow.

- Product browsing and search
- Cart management with AI assistance
- Checkout form with AI validation

### Dashboard Assistant

Admin dashboard with data tables, user management, and analytics.

- Interactive data tables
- User form with AI suggestions
- Stats cards and analytics

### Debug Assistant

Developer tools assistant with screenshot capture, console logging, and bug reporting.

- Screenshot capture and preview
- Console log viewer
- Bug report form with AI analysis

### Content Editor

Rich text editor with AI-powered content suggestions.

- Markdown editor
- Content preview
- AI publishing suggestions

## Getting Started

```bash
# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local
# Add your OPENAI_API_KEY to .env.local

# Run the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the showcase.
